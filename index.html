# About

Hello world! This website is just a mini-project that I created using HTML, CSS and Jquery. It only works when viewed on desktop as I have not included responsive web design in my code. Having no background knowledge in programming, it took me a period of 2 months to grasp the concepts, write the codes from scratch and stay up late at night to troubshoot annoying bugs. At the end of the day, all i can say is that the effort is worthwhile and coding is fun! The next step of my learning phase is to learn advance javascript and ultimately master solidity; the programming language for writing smart contracts in ethereum. I have a strong interest in Blockchain technology and hence this website is all about blockchain. Overtime, I will polish the website and add in new functions as I learn new languages. Meanwhile, please feel free to navigate around this simple webpage!
